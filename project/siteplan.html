W05 Website Planning Document

Site Name

Mineral Processing Equipment in Concentrator Plants

This name was selected because it clearly represents the focus of the website, which is mineral processing equipment used in mining plants.

Site Purpose

The purpose of this website is to provide clear and structured information about the main equipment used in mineral processing plants. It will help students and beginners understand how minerals are processed from raw ore into valuable concentrates through different stages.

Scenarios

What are the main machines used in a concentrator plant?

How does mineral processing work from crushing to flotation?

Color Schema

Dark Blue: used for headings and navigation bar

Gray: used for backgrounds and sections

Orange: used for highlights and buttons

Typography

Roboto: used for body text

Montserrat: used for headings

Wireframes

Mobile view: single column layout with stacked sections.

Desktop view: multi-column layout with navigation bar on top and content sections arranged in grid format.
